# currency-converter
A web-based currency converter built using HTML, CSS, and JavaScript.  
The application uses an external API to fetch real-time exchange rates and allows users to convert currencies instantly.

## Features
- Converts between different currencies
- Fetches real-time exchange rates using an API
- Simple and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Public Currency Exchange API

## What I Learned
- Working with APIs using JavaScript
- Fetching and handling JSON data
- DOM manipulation and event handling
